- 编写许多函数，并保持它们的小。在每一种功能中，做一件事并把它做好。

- 不要尝试使用regexp进行解析。它们不起作用，我建议把它们限制在非常简单的事情上。

- 不要试图猜测。当不确定如何解析某些内容时，抛出一个错误并确保消息包含错误位置(行/列)。
